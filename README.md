# Loan Approval Data Analysis

This project analyzes loan approval data to understand the key factors that affect loan approval decisions.

Dataset:
- Applicant income
- Co-applicant income
- Loan amount
- Credit history
- Loan term
- Property area

Tools Used:
- Excel / Python / SQL / Power BI

Key Analysis:
- Data cleaning and handling missing values
- Exploratory data analysis (EDA)
- Loan approval trends based on income and credit history
- Visualization of approval vs rejection

Insights:
- Applicants with good credit history have higher approval rates
- Income and loan amount significantly influence approval decisions

## Conclusion
This analysis helps understand how financial and demographic factors affect loan approval decisions and can assist financial institutions in better risk assessment.
